## Features

- **Use instantly** — No signup, no install. Open and use.
- **Download** — Get standalone HTML files that work offline.
- **Get the prompt** — Copy the AI prompt to customize or rebuild any tool.
- **AI Skills** — Add tools as reusable skills to your AI assistant *(coming soon)*.
- **Bilingual** — English and Spanish support.

---

## How It Works

Each tool has action buttons:

| Button | What it does |
|--------|--------------|
| **Use** | Open the tool in your browser |
| **Download** | Save as HTML file (works offline) |
| **Prompt** | Copy the AI prompt used to create it |
| **AI Skill** | Add to your AI assistant *(coming soon)* |
